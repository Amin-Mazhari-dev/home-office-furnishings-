# ROYAL CITY / home-office-furnishings

یک پروژه فروشگاهی کامل، ریسپانسیو و بدون استفاده از هیچ فریم‌ورک است. ساخته شده با HTML, CSS, JavaScript  Vanilla

## نمـــــــــــای کلـــــــــــــی پـــــــــــــــــروژه

 این یک پروژه فروشگاهی کامل و واقعی است که طی پنج ماه و با هدف تقویت مهارت‌های من در حوزه‌های زیر توسعه داده شده است:

- طراحی رابط کاربری ( UI )  
- تجربه کاربری ( UX )  
- ریسپانسیو حرفه‌ای  
- معماری فرانت‌اند  
- جاوااسکریپت پیور (  Vanilla JS) بدون هیچ فریم‌ورک  

در این پروژه از هیچ فریم‌ورکی مانند **Bootstrap** یا  **Tailwind**  استفاده نشده است.  
تمام بخش‌ها از صفر و با  **HTML ,  CSS  ,  JavaScript Vanilla **  طراحی و پیاده‌سازی شده‌اند.  
با وجود تسلط کامل به Bootstrap، این تصمیم کاملاً آگاهانه گرفته شد تا توانایی من در ساخت یک وب‌سایت واقعی بدون وابستگی به ابزارهای آماده سنجیده شود.

## مینــــــی فـــــــریــــم‌ورک اختصــــــاصــــی (  CSS)

در پوشه‌ی `assets/css` دو فایل مهم وجود دارد:

- **grid-system.css**  
  نوشته‌شده بر اساس اصول طراحی ریسپانسیو و ساختار شبکه‌ای ( Grid System )

- **utility-classes.css**  
  شامل مجموعه‌ای از کلاس‌های آماده برای استایل‌دهی سریع و استاندارد عناصر HTML

این دو فایل کاملاً توسط خودم نوشته شده‌اند و در کنار هم یک **مینی فریم‌ورک اختصاصی** برای طراحی ریسپانسیو تشکیل می‌دهند.

## صفحــــات پـــــــروژه

این پروژه شامل تمام بخش‌های یک فروشگاه واقعی است:

- صفحه اصلی  
- صفحه محصولات  
- صفحه جزئیات محصول  
- سبد خرید  
- ورود  
- ثبت‌نام  
- بازیابی رمز عبور  
- درباره ما  
- نمایندگی‌ها  
- و چندین بخش کاربردی دیگر  

## اهــــــداف پـــــــــــــــــروژه

- ساخت یک فروشگاه اینترنتی کامل بدون استفاده از فریم‌ورک‌هایی مانند Bootstrap یا Tailwind  

- تمرین و تقویت جاوااسکریپت پیور در سطح یک پروژه واقعی
  
- پیاده‌سازی ریسپانسیو کامل برای موبایل و دسکتاپ  

- طراحی UI/UX یکپارچه، تمیز و حرفه‌ای  

- مدیریت سبد خرید با منطق واقعی و قابل توسعه  

- ساخت صفحات کاربری (Login / Register / Recovery)  

- ایجاد ساختار یک وب‌سایت واقعی با هویت بصری و برند مشخص  

##  فنـــــــــاوری‌هـــــــــــای مـــــــورد استفـــــــاده

- ساختار اصلی صفحات  — **HTML5** 

- طراحی و استایل‌دهی کامل پروژه  —  **CSS3 (بدون فریم‌ورک)** 

- منطق، تعاملات و مدیریت رفتارهای پویا  — **Vanilla JavaScript** 

- مدیریت سبد خرید و ذخیره‌سازی داده‌های کاربر —  **LocalStorage** 

- **Responsive Design** — با استفاده از  Media Queries  و  Grid System اختصـــــــــاصی  


## طــــــــراحـــــــــی کاملاً واکنش‌گــــــــرا ( Responsive Design )

تمام صفحات پروژه برای سه حالت **موبایل، تبلت و دسکتاپ** به‌صورت کامل طراحی، پیاده‌سازی و تست شده‌اند.  
در این پروژه از  Media Queries، Grid System اختصــــــــاصی و کلاس‌های کمکی (Utility Classes) برای ایجاد یک تجربه‌ی ریسپانسیو روان و حرفه‌ای استفاده شده است.

##  ویــــــژگـــــــی‌هــــــای کلیـــــــــدی پــــــــــــروژه

### 1)  سبـــــــــــد خـــــــــرید (  Shopping Cart)
- افزودن محصول  
- حذف محصول  
- محاسبه مجموع قیمت  
- ذخیره‌سازی اطلاعات در LocalStorage  
- نسخه کامل برای موبایل و دسکتاپ  

### 2)  سیستم کاربری (User System)
- صفحه ورود ( Login )  
- صفحه ثبت‌نام با اعتبارسنجی ( Register + Validation )  
- صفحه بازیابی رمز عبور (Recovery)  
- پیام‌های خطا و راهنمای کاربر  

### 3) صفحات برند و اطلاعات مجموعه
- صفحه درباره ما  
- صفحه نمایندگی‌ها  
- اطلاعات کامل شعب  
- شبکه‌های اجتماعی  
- شماره تماس و آدرس  

### 4) صفحــــــات محصــــــول
- صفحه اصلی  
- دسته‌بندی محصولات  
- صفحه جزئیات محصول  
- تصاویر باکیفیت  
- طراحی یکپارچه و هماهنگ در تمام صفحات
